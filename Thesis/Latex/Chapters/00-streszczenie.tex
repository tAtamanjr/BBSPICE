\chapter*{Streszczenie}

Praca dotyczy projektu i implementacji prostego symulatora ukladow elektronicznych wykorzystujacego metode Modified Nodal Analysis. Program przetwarza tekstowy opis obwodu, tworzy zbior stempli elementow, sklada macierz ukladu i rozwiazuje punkt pracy dla ukladow liniowych.

\vspace{4mm}
\noindent\textbf{Slowa kluczowe:} Modified Nodal Analysis, symulacja obwodow, SPICE, Swift, metody numeryczne.

\chapter*{Abstract}

The thesis describes the design and implementation of a simple electronic circuit simulator based on Modified Nodal Analysis. The program parses a textual circuit description, creates element stamps, assembles the system matrix, and solves the operating point for linear circuits.

\vspace{4mm}
\noindent\textbf{Keywords:} Modified Nodal Analysis, circuit simulation, SPICE, Swift, numerical methods.
