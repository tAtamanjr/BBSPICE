\chapter*{Streszczenie}

Praca inżynierska skupia się na napisaniu dedykowanej aplikacji do analizy analogowych układów elektronicznych w języku programowania Swift. Aplikacja stanowi próbę pokazania możliwości tworzenia natywnych narzędzi do analizy układów elektronicznych dla systemu macOS. Prace rozpoczęto od zebrania informacji dotyczących sposobów tworzenia aplikacji służących do analizy układów elektronicznych, metod numerycznych pozwalających na ich analizę oraz pogłębienia znajomości języka programowania Swift. Zakres pracy obejmował napisanie poszczególnych bloków aplikacji umożliwiających parsowanie opisu układu ze zwykłego pliku tekstowego, tworzenie elementów na podstawie ich szablonów, obliczanie punktu pracy i analizę czasową układu przy użyciu metod numerycznych. W procesie zostały napisane zabezpieczenia chroniące przed analizowaniem błędnych układów oraz testy sprawdzające poszczególne bloki aplikacji. Poprawność opracowanego rozwiązania została sprawdzona na prostych układach oraz na przykładzie wtórnika emiterowego.

\vspace{4mm}
\noindent\textbf{Słowa kluczowe:} ZMPW, symulacja obwodów, SPICE, Swift, metody numeryczne.

\chapter*{Abstract}

The engineering thesis focuses on developing a dedicated application for the analysis of analog electronic circuits in the Swift programming language. The application is intended to demonstrate the possibility of creating native circuit analysis tools for macOS. The work began with gathering information on applications used for electronic circuit analysis, numerical methods applied to such analysis, and the Swift programming language. The scope of the work included implementing the main application modules responsible for parsing a circuit description from a plain text file, creating elements based on their templates, calculating the operating point, and performing time-domain analysis using numerical methods. During the development process, safeguards were implemented to prevent the analysis of incorrect circuits, and tests were prepared to verify individual application modules. The correctness of the developed solution was checked on simple circuits and on an emitter follower example.

\vspace{4mm}
\noindent\textbf{Keywords:} MNA, circuit simulation, SPICE, Swift, numerical methods.
